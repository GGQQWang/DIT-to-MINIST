# 第一阶段：类别加噪子空间几何假设验证

日期：2026年6月10日

## 9.1 实验目标

第一阶段不训练模型，只验证“类别中心 + 加噪点”是否真的可以形成有效分类子空间。

本阶段希望回答以下问题：

1. 类别加噪子空间是否存在稳定低维结构；
2. 子空间投影残差是否具有分类能力；
3. 不同噪声强度对子空间稳定性有何影响；
4. 子空间维度 `d` 如何影响分类效果；
5. 中等噪声是否优于低噪声和高噪声。

## 研究逻辑

已有 DiT 原型收缩实验表明，在中等噪声条件下，模型将加噪图像向类别平均原型收缩后，分类效果显著优于直接使用原始图像进行原型匹配。

这一现象提出了一个更基础的问题：

```txt
DiT 的有效性是否来自加噪空间中本身存在的类别几何结构？
```

因此，第一阶段不训练任何模型，而是在有监督条件下使用真实标签构造类别中心和类别加噪子空间，验证这个几何结构是否真实存在、是否具有判别能力、是否对噪声强度和子空间维度稳定。

本阶段的定位是：

```txt
有监督几何验证阶段
```

它不是最终的无监督聚类方法，而是为后续生成式聚类提供几何前提。其核心逻辑为：

```txt
Motivation:
已有 DiT 原型收缩实验显示，中等噪声下的类别原型收缩能显著提升分类效果。

Question:
这种提升是否来自加噪空间本身存在的类别子空间结构？

Stage 1:
在真实标签已知的理想条件下，构造类别中心和类别加噪子空间，检验其存在性、判别性和稳定性。

Future:
如果该结构稳定存在，则进一步研究无标签条件下如何自动发现这些中心和子空间，并用 DiT 学习对应的吸引域收缩映射。
```

因此，第一阶段回答的是一个上限问题：

```txt
如果类别归属已知，类别中心 + 加噪样本能否形成有效子空间？
```

如果答案是否定的，则后续无监督寻找该结构没有意义。  
如果答案是肯定的，则说明该结构值得进一步发展为无监督生成式聚类方法。

## 核心假设

对于每个类别 `c`，先由训练样本得到类别中心：

```txt
mu_c = mean(x | y = c)
```

然后对该类别样本进行加噪：

```txt
x_t = sqrt(alpha_bar_t) x_0 + sqrt(1 - alpha_bar_t) eps
```

如果流形假说在当前任务中成立，那么同一类别的加噪样本不应完全变成无结构的随机点云，而应在某些噪声强度下仍保留稳定的低维几何结构。

由于加噪公式会把均值按 `sqrt(alpha_bar_t)` 缩放，因此在加噪空间中应使用同步缩放后的类别中心：

```txt
mu_c(t) = sqrt(alpha_bar_t) mu_c
```

因此，可以围绕加噪类别中心构造一个局部子空间：

```txt
S_c(t, d) = mu_c(t) + Span(v_c1(t), v_c2(t), ..., v_cd(t))
```

其中 `v_ci` 可以由该类别加噪样本相对类别中心的 PCA 主方向得到。

## 验证思路

对测试样本 `x`，在同一噪声强度 `t` 下得到 `x_t`，然后分别计算它到每个类别子空间的投影残差：

```txt
r_c(x_t) = ||x_t - Proj_{S_c(t,d)}(x_t)||^2
```

等价地：

```txt
r_c(x_t) = ||(I - V_c(t,d)V_c(t,d)^T)(x_t - mu_c(t))||^2
```

如果最小残差对应真实类别：

```txt
pred = argmin_c r_c(x_t)
```

则说明“类别中心 + 加噪点”构造出的子空间具有分类能力。

## 实验变量

本阶段主要控制两个变量：

```txt
噪声强度 t
子空间维度 d
```

噪声强度可以设置为：

```txt
t ∈ {0, 50, 100, 150, 200, 300, 500}
```

子空间维度可以设置为：

```txt
d ∈ {1, 2, 5, 10, 20, 50, 100}
```

## 评价指标

1. 分类准确率：

```txt
acc(t, d)
```

用于判断投影残差是否真的有分类能力。

2. 类内投影解释率：

```txt
explained_variance_c(t, d)
```

用于衡量类别加噪点是否存在稳定低维结构。

3. 类间残差间隔：

```txt
margin = min_wrong_residual - true_class_residual
```

用于判断正确类别子空间是否和错误类别子空间有清晰分离。

4. 噪声强度敏感性：

比较不同 `t` 下的分类准确率和 margin，验证是否存在中等噪声最优区间。

5. 子空间方向稳定性：

对同一个 `t` 和 `d`，使用不同噪声随机种子重新构造类别子空间，比较子空间主角相似度：

```txt
sim(V_a, V_b) = ||V_a^T V_b||_F^2 / d
```

该值越接近 1，说明类别加噪子空间越稳定。

## 预期判断

如果实验结果显示：

```txt
中等 t 下 acc 较高
中等 t 下 margin 较大
过低 t 时类内个体差异过强
过高 t 时类别结构被噪声破坏
适中的 d 能达到较好效果，过大 d 反而引入噪声方向
```

则可以支持以下判断：

```txt
类别中心 + 加噪样本确实能够形成具有分类意义的局部子空间。
```

这将为后续无标签生成式聚类提供几何依据：不直接在原始图像流形上聚类，而是在适度加噪后的空间中寻找稳定中心及其局部子空间。
