## 2. 主实验：不同模型，同一中等噪声，一步推理

先固定 `train_timestep=200`，比较模型学习原型吸引场的能力：

```bash
for model in linear mlp cnn cnn-strong unet dit-xs dit-s dit-b
do
  python attraction_field_models.py \
    --device cuda \
    --model $model \
    --epochs 100 \
    --batch-size 128 \
    --eval-batch-size 256 \
    --train-per-class 0 \
    --val-per-class 1000 \
    --test-per-class 0 \
    --train-timestep 200 \
    --eval-timesteps 200 \
    --noise-repeats 3 \
    --val-noise-repeats 1 \
    --inference-steps 1 \
    --lr 1e-4 \
    --patience 10 \
    --num-workers 8 \
    --output-dir ./吸引场验证/outputs_t200_${model}
done
```

重点比较：

```txt
val_metrics.csv
test_metrics.csv
train_log.csv
summary.md
```

## 3. 多步吸引推理

多步版本不改变训练目标，仍然训练：

```txt
x_t -> mu_y
```

但推理时重复执行：

```txt
p_hat = model(x, t)
x <- x + eta * (p_hat - x)
```

先只比较强模型：

```bash
for model in cnn-strong unet dit-xs dit-s dit-b
do
  python attraction_field_models.py \
    --device cuda \
    --model $model \
    --epochs 100 \
    --batch-size 128 \
    --eval-batch-size 256 \
    --train-per-class 0 \
    --val-per-class 1000 \
    --test-per-class 0 \
    --train-timestep 200 \
    --eval-timesteps 200 \
    --noise-repeats 3 \
    --val-noise-repeats 1 \
    --inference-steps 5 \
    --attraction-step-size 0.4 \
    --inference-schedule constant \
    --lr 1e-4 \
    --patience 10 \
    --num-workers 8 \
    --output-dir ./吸引场验证/outputs_t200_multistep5_${model}
done
```

可选测试 decreasing timestep：

```bash
for model in cnn-strong unet dit-b
do

    --test-per-class 0 \
    --train-timestep 200 \
    --eval-timesteps 200 \
    --noise-repeats 3 \
    --val-noise-repeats 1 \
```

## 4. 噪声强度消融

比较不同训练噪声强度：

```bash
for t in 0 100 200 300
do
  for model in linear mlp cnn cnn-strong unet dit-xs dit-s dit-b
  do
    python attraction_field_models.py \
      --device cuda \
      --model $model \
      --epochs 100 \
      --batch-size 128 \
      --eval-batch-size 256 \
      --train-per-class 0 \
      --val-per-class 1000 \
      --test-per-class 0 \
      --train-timestep $t \
      --eval-timesteps $t \
      --noise-repeats 3 \
      --val-noise-repeats 1 \
      --inference-steps 1 \
      --lr 1e-4 \
      --patience 10 \
      --num-workers 8 \
      --output-dir ./吸引场验证/outputs_t${t}_${model}
  done
done
```

判断重点：

```txt
哪些模型在 t=200 或 t=300 优于 t=0；
Linear 是否明显弱于非线性模型；
强 CNN 是否明显强于小 CNN；
U-Net 是否强于 CNN-Strong 和 DiT；
DiT-XS/DiT-S/DiT-B 是否随规模提升；
是否只有 DiT 呈现中等噪声最优。
```

## 6. 输出文件

每次运行会生成：

```txt
train_log.csv       每个 epoch 的训练 loss
val_metrics.csv     每个 epoch 的 validation acc、margin、距离、收缩比例
test_metrics.csv    best validation checkpoint 在 test set 上的 acc、margin、距离、收缩比例
summary.md          最终摘要
best_checkpoint.pt  validation 最优模型权重
final_checkpoint.pt 已加载 best 权重后的最终保存
```

## 7. 优先级建议

如果服务器时间有限，先跑：

```txt
linear
mlp
cnn-strong
unet
dit-xs
dit-b
```

并优先比较：

```txt
t=0
t=200
```

如果 `dit-xs` 已经接近 `dit-b`，可以减少后续 DiT-B 大规模实验。
