# ProtoNet 辅助原型去噪模块设计

## 1. 原始 ProtoNet 框架

在原始 ProtoNet 中，support 样本和 query 样本首先经过同一个编码器：

\[
z=f_\theta(x)
\]

对于每个类别 \(k\)，由 support set 中该类样本的 embedding 取均值得到类别原型：

\[
c_k=\frac{1}{K}\sum_{i=1}^{K} f_\theta(x_i^k)
\]

其中，\(K\) 表示每类 support 样本数量。

query 样本经过编码器后得到特征表示：

\[
z_q=f_\theta(x_q)
\]

随后计算 query embedding 与各类别 prototype 的距离，并通过 softmax 得到分类概率：

\[
P(y=k|x_q)=
\frac{\exp(-d(z_q,c_k))}
{\sum_j \exp(-d(z_q,c_j))}
\]

原始 ProtoNet 的分类损失为：

\[
L_{\mathrm{proto}}
=
-\log P(y|x_q)
\]

---

## 2. 辅助原型去噪模块

为了增强 ProtoNet 在特征扰动条件下的原型收缩能力，本文在原始分类损失之外加入辅助原型去噪模块。

首先，在 query embedding 上加入高斯扰动：

\[
\tilde z_q=z_q+\sigma\epsilon
\]

其中：

\[
\epsilon\sim\mathcal N(0,I)
\]

\(\sigma\) 表示噪声强度。

---

## 3. 噪声强度量化方式

由于 embedding 空间的数值尺度并不固定，本文不直接使用固定噪声强度，而是采用相对噪声强度进行控制：

\[
\sigma=\rho\cdot \operatorname{Std}(z_q)
\]

其中，\(\rho\) 是噪声比例系数，用于控制噪声标准差占当前 query embedding 标准差的比例。

例如：

\[
\rho\in\{0.05,0.1,0.2,0.5,1.0\}
\]

当 \(\rho=0.2\) 时，表示加入的噪声标准差约为当前特征标准差的 20%。

也可以使用特征空间信噪比描述噪声强度：

\[
\mathrm{SNR}_z=
\frac{\mathbb E\|z_q\|_2^2}
{\sigma^2 d}
\]

其中，\(d\) 为 embedding 维度。

---

## 4. 两层残差 MLP 去噪模块

辅助模块采用轻量级两层残差 MLP。设输入为扰动后的 query embedding \(\tilde z_q\)，则去噪模块定义为：

\[
h_\phi(\tilde z_q)
=
\tilde z_q+
W_2\mathrm{ReLU}(W_1\tilde z_q+b_1)+b_2
\]

其中：

- 输入维度为 \(d\)；
- 隐藏层维度可取 \(2d\)；
- 输出维度仍为 \(d\)；
- 残差结构用于让模块学习扰动特征的修正量，而不是完全重写 embedding。

得到去噪后的 query 表示：

\[
\hat z_q=h_\phi(\tilde z_q)
\]

---

## 5. 辅助原型回归损失

对于 query 样本，其真实类别为 \(y\)。正确类别原型 \(c_y\) 由当前 episode 中 support set 的同类 embedding 均值得到：

\[
c_y=\frac{1}{K}\sum_{i=1}^{K} f_\theta(x_i^y)
\]

为了使扰动 query 特征回到正确类别原型中心，定义辅助原型回归损失：

\[
L_{\mathrm{denoise}}
=
\|\hat z_q-\mathrm{sg}(c_y)\|_2^2
\]

其中，\(\mathrm{sg}(\cdot)\) 表示 stop-gradient 操作。

stop-gradient 的作用是：

- 将 prototype \(c_y\) 作为固定监督目标；
- 避免辅助损失反向牵动 support prototype；
- 但梯度仍然可以通过 query 分支传回 MLP 和 encoder。

也就是说，辅助损失的梯度路径为：

\[
L_{\mathrm{denoise}}
\rightarrow
\hat z_q
\rightarrow
\tilde z_q
\rightarrow
z_q
\rightarrow
f_\theta
\]

而不会通过：

\[
L_{\mathrm{denoise}}
\rightarrow
c_y
\rightarrow
z_s
\rightarrow
f_\theta
\]

---

## 6. 总体训练目标

最终模型的训练目标由原始 ProtoNet 分类损失和辅助原型去噪损失共同组成：

\[
L=
L_{\mathrm{proto}}
+
\lambda L_{\mathrm{denoise}}
\]

其中，\(\lambda\) 用于控制辅助损失的权重。

---

## 7. 方法核心思想

该模块并不是直接将加噪特征用于原型距离分类，而是通过显式的原型回归约束，使扰动后的 query 特征经过 MLP 后回到正确类别 prototype 附近。

原始 ProtoNet 的目标主要是：

\[
d(z_q,c_y)<d(z_q,c_k),\quad k\neq y
\]

即保证 query 更接近正确类别原型。

而本文加入的辅助模块进一步约束：

\[
h_\phi(z_q+\sigma\epsilon)\rightarrow c_y
\]

即要求扰动后的 query embedding 能够被映射回正确类别原型中心。

因此，该方法可以被理解为一种 prototype-guided denoising regularization，用于增强 ProtoNet 特征空间在扰动条件下的稳定性和原型收缩能力。

---

## 8. 论文表述版本

本文在 ProtoNet 的原型度量分类框架中引入辅助原型去噪模块。该模块并不直接将加噪特征用于距离分类，而是在 query embedding 上施加尺度自适应高斯扰动，并通过轻量级两层残差 MLP 将扰动特征回归到其真实类别 prototype。具体而言，support 样本经编码器后按类别取均值构造原型 \(c_k\)，query 样本经编码器得到 \(z_q\)，随后加入噪声得到 \(\tilde z_q=z_q+\sigma\epsilon\)。扰动强度由 \(\sigma=\rho\cdot\operatorname{Std}(z_q)\) 控制，从而使噪声大小与当前特征尺度保持一致。去噪模块 \(h_\phi(\cdot)\) 采用两层残差 MLP，输出 \(\hat z_q=h_\phi(\tilde z_q)\)。辅助损失定义为 \(L_{\mathrm{denoise}}=\|\hat z_q-\mathrm{sg}(c_y)\|_2^2\)，其中 \(c_y\) 为正确类别 support prototype，stop-gradient 操作用于保证 prototype 仅作为监督目标。最终模型以 \(L=L_{\mathrm{proto}}+\lambda L_{\mathrm{denoise}}\) 进行联合优化。该设计通过显式约束扰动 query 特征向正确类别原型收缩，增强了特征空间在噪声扰动下的稳定性和原型判别能力。
